# Practical Exercise 3


## How to run project
1. Run with make(**Recommended**)

    `
        make
    `


2. Compile and run with Command Line.

    `
        gcc src/core/flush.c src/core/linkedlist.c -o flush
    `

    `
        ./flush
    `
